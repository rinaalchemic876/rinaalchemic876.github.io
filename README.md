# ryot3827.github.io

GitHub Pages で公開する個人経歴書（ポートフォリオ）サイトです。

## 🌐 公開 URL

https://ryot3827.github.io

## 📁 ファイル構成

```
.
├── index.html   # メインページ（経歴書）
├── style.css    # スタイルシート
└── README.md    # このファイル
```

## ✏️ カスタマイズ方法

`index.html` を編集して自分の情報に書き換えてください。

| セクション | 編集箇所 |
|----------|---------|
| 名前・肩書き | `<h1>` タグ内 |
| 連絡先・SNS | `profile-contact` 内のリンク |
| 自己紹介 | `summary` セクション |
| スキル | `skills-grid` 内の `tag` 要素 |
| 職務経歴 | `timeline` （職務経歴セクション）|
| 学歴 | `timeline` （学歴セクション）|
| 個人プロジェクト | `projects-grid` 内の `project-card` |
| 資格・認定 | `timeline` （資格セクション）|

## 🚀 GitHub Pages の有効化

1. このリポジトリの **Settings** → **Pages** を開く
2. **Source** を `Deploy from a branch` に設定
3. **Branch** を `main`（または `master`）、フォルダを `/ (root)` に設定
4. **Save** をクリック

数分後に `https://ryot3827.github.io` で公開されます。
